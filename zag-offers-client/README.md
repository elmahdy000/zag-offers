# Zag Offers Client Web App

تطبيق الويب الرسمي لمستخدمي زقازيق أوفرز، مبني باستخدام Next.js 15 و Tailwind CSS.

## المميزات
- تصفح أحدث العروض والخصومات في مدينة الزقازيق.
- بحث متقدم وفلترة حسب المناطق (الجامعة، القومية، وسط البلد) والأقسام.
- نظام استخراج الكوبونات الحصرية.
- إدارة المفضلة والكوبونات الشخصية.
- تصميم عصري (Dark Mode) يدعم الموبايل والتابلت والكمبيوتر.

## التشغيل المحلي
```bash
npm install
npm run dev
```

## المتطلبات
- Node.js 18+
- Backend API (api.zagoffers.online)
